# NicochViewerUWP
ニコニコチャンネルの自動キャッシュサーバー[nicochcgi](https://github.com/kurema/nicochcgi_docker)専用クライアントです。

このアプリを利用するには、上記サービスをLAN内のサーバーにインストールする事が必要です。
